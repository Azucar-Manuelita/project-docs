# Merequetengue

## by Azucar Manuelita

**ncaucali@unal.edu.co (Nicolas Caucali)**



**Descripción**

Merequetengue es una pagina web que proporciona tutorías automatizadas y seguimiento a los usuarios inscritos en un gimnasio. Dicha tutoría esta basada en su estado físico inicial, objetivos  personales y periodo de entrenamiento definido. El sistema busca mejorar la experiencia de usuario mediante la asignación de sesiones de entrenamiento personalizadas, para optimizar el uso del tiempo dentro del gimnasio y apoyar a los instructores reduciendo su carga sobre la planificación.
